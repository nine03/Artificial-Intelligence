# 利用三层神经网络与信号传递过程(삼층신경망과 신호전달과정)

### 主要原理(주요원리)
我用三层神经网络为对象，实现从输入到输出的处理。使用Numpy数组，可以完成神经网路的前向处理。(삼층신경망을 대상으로 입력에서 출력까지의 처리를 실현하고 Numpy 함수를 이용하여 신경망으로의 전방향 처리가 할수있다.)
